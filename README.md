# 🎧 Sonicstride — Smart AI DJ

An AI DJ for well-being. It reads your current moment (energy, sleep, stress, weather,
schedule, time of day), reasons like a DJ who cares, predicts the ideal sound, and plays
real, mood-matched Spotify songs.

```bash
cp .env.example .env     # add OPENAI_API_KEY (+ Spotify creds)
npm install
npm start                # http://127.0.0.1:3000
```

📖 **Everything is in [`HANDOFF.md`](./HANDOFF.md)** — product vision, architecture, the
AI design, verified Spotify constraints, the full API contract for the frontend, env
vars, and the Phase-2 roadmap. Start there.
